# Installation

```bash
bun install
```

## Run the project

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```

```bash
# build application
$ bun run build
```

```bash
bun run dist/main.js
```

```bash
# unit tests
$ bun run test

# e2e tests
$ bun run test:e2e

# test coverage
$ bun run test:cov
```
